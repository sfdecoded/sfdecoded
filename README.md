## Salesforce Decoded

Salesforce Decoded is a collection of tools, guides, and troubleshooting resources built to help Salesforce developers work faster and solve problems more efficiently.

The platform focuses on practical utilities and documentation related to Apex, SOQL, Lightning development, debugging, and developer productivity.

Live site: https://sfdecoded.netlify.app

⸻

**Overview**

Developing on the Salesforce platform often involves repetitive tasks such as formatting queries, debugging governor limits, generating Apex structures, and troubleshooting platform errors.

Salesforce Decoded was created to provide a centralized hub for:
	•	Developer productivity tools
	•	Salesforce error explanations and fixes
	•	Implementation guides and best practices
	•	Development tips for Apex, SOQL, and Lightning

The objective is to simplify everyday development workflows while providing high-quality technical resources.

⸻

**Features**

**Developer Tools**

Salesforce Decoded includes utilities that simplify common development tasks.

Examples include:
	•	SOQL Formatter
	•	Salesforce ID Converter
	•	JSON to Apex Converter
	•	Apex Code Generator
	•	Salesforce Icon Explorer
	•	Field Dependency Viewer

These tools help developers quickly perform tasks that would otherwise require manual effort.

⸻

**Error Reference Library**

Salesforce errors can often be difficult to understand. Salesforce Decoded provides explanations and solutions for common platform errors such as:
	•	Too Many SOQL Queries (101)
	•	Apex CPU Time Limit Exceeded
	•	Insufficient Access Cross Reference Entity
	•	Unable to Lock Row

Each error page explains the root cause and recommended strategies to resolve the issue.
